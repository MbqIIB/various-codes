http://content.googleapis.com/content/v1/$CID/items/products/schema/
e.g.
 http://content.googleapis.com/content/v1/123456/items/products/schema/


For updating/deleting/describing offers:
http://content.googleapis.com/content/v1/$CID/items/products/schema/online:$LANGUAGE:$COUNTRY:$ITEMID
e.g.
 http://content.googleapis.com/content/v1/123456/items/products/schema/online:en:US:ID001

For batch requests -
http://content.googleapis.com/content/v1/$CID/items/products/schema/batch
e.g.
http://content.googleapis.com/content/v1/123456/items/products/schema/batch

When an item is created, one of the attributes in the returned XML is a "<link rel='edit'..." element. All batch requests that update or delete an item must include this rel="edit" link - e.g.
<link rel='edit' type='application/atom+xml' href='http://content.googleapis.com/content/v1/123456/items/products/schema/online:en:US:ID001/>

For MCA requests
http://content.googleapis.com/content/v1/$CID/managedaccounts/products
http://content.googleapis.com/content/v1/$CID/managedaccounts/products/$SUB_ACT

e.g.

http://content.googleapis.com/content/v1/123456/managedaccounts/products
http://content.googleapis.com/content/v1/123456/managedaccounts/products/222222

The old URLs will continue to work until the end of the year. Please let me know if you have any questions about this.
