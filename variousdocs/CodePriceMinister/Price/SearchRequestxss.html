 Description   
Un formulaire de contact dédié à la relation entre les clients du site PriceMinister et la société PriceMinister est accessible depuis la section « FAQ/Contact » du site http://www.priceminister.com

Les deux champs « Objet » et « Votre Message » sont vulnérables aux attaques de type Cross Site Scripting, plus communément appelées XSS. Une page est vulnérable au XSS lorsque celle-ci retourne une page de résultats contenant, au caractère près, le motif recherché. Dans ce cas, le motif recherché sera du code HTML ou Javascript qui sera alors évalué par le serveur web

Par exemple il est possible d'insérer le motif <script>alert(document.cookie)</script> dans le champ « Votre Message ». La commande Javascript document.cookie permet de récupérer entre autres le cookie de session de l'internaute dans le but d'usurper son identité auprès du site web.
